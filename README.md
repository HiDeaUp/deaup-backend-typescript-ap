# Backend TypeScript API

## Installation

```bash
$ yarn install
```


### Setup
1. Rename `.env.dist` file to `.env`
2. Then, open `.env` and update the database credentials with yours.


## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```


## Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
